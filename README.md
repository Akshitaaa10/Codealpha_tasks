# CodeAlpha Frontend Development Internship

This repository contains the tasks completed as part of the **CodeAlpha Frontend Developer Internship**. Each task is a self-contained, single-file HTML application built using HTML, CSS, and JavaScript — no external frameworks required.

## 📁 Tasks

### 1. Calculator — `CodeAlpha_Calculator.html`
A responsive calculator supporting basic arithmetic operations (addition, subtraction, multiplication, division) with a clean, minimal interface.

**Features:**
- Real-time expression display
- Keyboard and click support
- Clear/delete functionality
- Responsive layout for desktop and mobile

---

### 2. Music Player — `CodeAlpha_MusicPlayer.html`
A mixtape-style music player with a custom dark, cassette-inspired UI. Supports multiple mixtapes/albums, a persistent player bar, search, and liked tracks.

**Features:**
- Sidebar navigation with multiple mixtapes/albums
- Custom play/pause, next/previous, shuffle, and repeat controls
- Seek bar and volume control
- Global search across all mixtapes (by track, artist, or album name)
- Liked Tracks view — like any track and revisit it from a dedicated tab
- Responsive design for smaller screens
- Keyboard shortcut: spacebar to play/pause

**Note:** Audio `src` fields are intentionally left empty in the source code to avoid copyright issues with commercial tracks. To test playback, add your own royalty-free audio file links inside the `ALBUMS` array in the `<script>` section.

---

### 3. Image Gallery — `CodeAlpha_ImageGallery.html`
A minimal, responsive image gallery widget for browsing and viewing a collection of images.

**Features:**
- Grid-based responsive layout
- Smooth hover/transition effects
- Lightweight, dependency-free implementation

---

## 🛠️ Tech Stack
- HTML5
- CSS3 (custom properties, Grid, Flexbox)
- Vanilla JavaScript (no libraries/frameworks)
- Google Fonts (used in Music Player)

## 🚀 How to Run
No build steps or installations required.

1. Clone or download this repository.
2. Open any of the `.html` files directly in a browser:
   - `CodeAlpha_Calculator.html`
   - `CodeAlpha_MusicPlayer.html`
   - `CodeAlpha_ImageGallery.html`
